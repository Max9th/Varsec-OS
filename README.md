## 🔧 Installation & Setup  

### 📥 Cloning the Repository  
```sh
git clone https://github.com/Max9th/Varsec-OS.git
cd Varsec-OS
```

### 🎮 Running the Project in Godot  
1. Open **Godot v4.4**  
2. Click **Import Project**  
3. Select the `project.godot` file in the repository  e 
4. Click **Run**
